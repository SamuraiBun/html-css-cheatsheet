HTML Elements Cheatsheet!

Table Elements

Tag Name Description
<table> Table Defines a table
<th> Table Header Defines a header cell in a table
<tr> Table Row Defines a row in a table
<td> Table Data Defines a cell in a table
<caption> Table Caption Defines a table caption
<colgroup> Column Group Specifies a group of one or more columns in a table for formatting
<col> Column Properties Specifies column properties for each column within a <colgroup> element
<thead> Table Header Groups the header content in a table
<tbody> Table Body Groups the body content in a table
<tfoot> Table Footer Groups the footer content in a table

Table Attributes

Attribute Name Description
colspan Column Span Specifies the number of columns a cell should span
rowspan Row Span Specifies the number of rows a cell should span
headers Header Cells Specifies one or more header cells a cell is related to
scope Header Cell Scope Specifies whether a header cell is a header for a column, row, or group of columns or rows

Form Elements

Tag Name Description
<form> Form Defines an HTML form for user input
<input> Input Defines an input control
<textarea> Text Area Defines a multiline input control (text area)
<button> Button Defines a clickable button
<select> Select Defines a drop-down list
<optgroup> Option Group Defines a group of related options in a drop-down list
<option> Option Defines an option in a drop-down list
<fieldset> Field Set Groups related elements in a form
<legend> Legend Defines a caption for a <fieldset> element
<label> Label Defines a label for an <input> element

Form Attributes

Attribute Name Description
autocomplete Auto Complete Specifies whether a form or input field should have autocomplete enabled
novalidate No Validate Specifies that the form should not be validated when submitted
form Form Specifies the form the input element belongs to
formaction Form Action Specifies the URL where the form data should be submitted
formenctype Form Encoding Type Specifies how form data should be encoded before sending it to a server
formmethod Form Method Specifies the HTTP method to use when sending form data
formnovalidate Form No Validate Specifies that the form should not be validated when submitted
formtarget Form Target Specifies where to display the response after submitting the form
height Height Specifies the height of an <input> element
list List Refers to a <datalist> element that contains pre-defined options for an <input> element

Tag Name Description

<b> Bold Text Defines bold text

Text Attributes

Attribute Name Description
accesskey Access Key Specifies a shortcut key to activate/focus an element
contenteditable Content Editable Specifies whether the content of an element is editable or not
contextmenu Context Menu Specifies a context menu for an element
dir Direction Specifies the text direction for the content in an element
draggable Draggable Specifies whether an element is draggable or not
dropzone Drop Zone Specifies whether the dragged data is copied, moved, or linked, when dropped
hidden Hidden Specifies that an element is not yet, or is no longer, relevant
lang Language Specifies the language of the element's content
spellcheck Spell Check Specifies whether the element is to have its spelling and grammar checked or not
title Title Specifies extra information about an element

Tag Name Description

<area> Image Map Area Defines a clickable area inside an image map
<canvas> Canvas Defines graphics
<figcaption> Figure Caption Defines a caption for a <figure> element
<figure> Figure Specifies self-contained content
<picture> Picture Defines a container for multiple image resources
<source> Media Resources Defines multiple media resources for media elements (<video> and <audio>)
<track> Text Track Defines text tracks for media elements (<video> and <audio>)

Image Attributes

Attribute Name Description

height Height Specifies the height of an image
ismap Image Map Specifies an image as a server-side image map
longdesc Long Description Specifies a URL to a detailed description of an image
sizes Image Sizes Specifies the sizes of the icons for different page layouts
src Source Specifies the URL of an image
srcset Source Set Specifies the URL of the image to use in different situations
usemap Use Map Specifies an image as a client-side image map
width Width Specifies the width of an image

Link Elements

Tag Name Description
<a> Anchor Defines a hyperlink
<link> Link Defines the relationship between a document and an external resource
<nav> Navigation Defines navigation links
<link> Link Defines the relationship between a document and an external resource
<nav> Navigation Defines navigation links
<menu> Menu Defines a list/menu of commands
<menuitem> Menu Item Defines a command/menu item that the user can invoke from a popup menu

Link Attributes

Attribute Name Description
download Download Specifies that the target will be downloaded when a user clicks on the hyperlink
href Hyperlink Reference Specifies the URL of the page the link goes to
hreflang Hyperlink Language Specifies the language of the linked document
media Media Type Specifies what media/device the linked document is optimized for
ping Link Ping Specifies a space-separated list of URLs to which, when the link is followed, post requests with the body ping will be sent by the browser
rel Relationship Specifies the relationship between the current document and the linked document
target Target Specifies where to open the linked document
type Media Type Specifies the media type of the linked document

Meta Elements

Tag Name Description
<base> Base Specifies the base URL/target for all relative URLs in a document
<head> Head Defines information about the document
<link> Link Defines the relationship between a document and an external resource
<meta> Meta Defines metadata about an HTML document
<style> Style Defines style information for a document
<title> Title Defines a title for the document

Meta Attributes

Attribute Name Description
charset Character Set Specifies the character encoding for the HTML document
content Meta Content Specifies the value associated with the http-equiv or name attribute
http-equiv HTTP Equivalent Provides an HTTP header for the information/value of the content attribute
name Name Specifies the name of the metadata
scheme Metadata Scheme Specifies a scheme to be used to interpret the value of the content attribute

Script Elements

Script Attributes

Attribute Name Description

src Source Specifies the URL of an external script file
type Type Specifies the media type of the script

Flexbox Properties

Property Name Description

align-items Align Items Specifies the default alignment for items inside the flexible container
align-self Align Self Specifies the alignment for selected items inside the flexible container
flex Flex Specifies the length of the item, relative to the rest

flex-direction Flex Direction Specifies the direction of the flexible items

flex-wrap Flex Wrap Specifies whether the flexible items should wrap or not
justify-content Justify Content Specifies the alignment between the items inside a flexible container when the items do not use all available space
order Order Specifies the order of the flexible item

Animation Properties

Property Name Description
animation Animation Shorthand property for setting all the animation properties in one declaration
animation-delay Animation Delay Specifies when the animation will start
animation-direction Animation Direction Specifies whether the animation should play in reverse on alternate cycles
animation-duration Animation Duration Specifies how many seconds or milliseconds an animation takes to complete one cycle
animation-fill-mode Animation Fill Mode Specifies a style for the element when the animation is not playing (before it starts, after it ends, or both)
animation-iteration-count Animation Iteration Count Specifies the number of times an animation should be played
animation-name Animation Name Specifies a name for the @keyframes animation
animation-play-state Animation Play State Specifies whether the animation is running or paused
animation-timing-function Animation Timing Function Specifies the speed curve of the animation

Transition Properties

Property Name Description

transition-delay Transition Delay Specifies when the transition effect will start
transition-duration Transition Duration Specifies how many seconds or milliseconds a transition effect takes to complete
transition-property Transition Property Specifies the name of the CSS property the transition effect is for
transition-timing-function Transition Timing Function Specifies the speed curve of the transition effect

Transform Properties
